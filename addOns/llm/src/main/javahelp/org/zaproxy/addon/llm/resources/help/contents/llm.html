LLM Support

This add-on integrates LLM with ZAP and includes two main features: API Sequencing: Import OpenAPI definitions to generate sequences of HTTP calls for subsequent scanning operations. Alert Review: Examine an alert and determine the confidence level based on evidence from ZAP, complete with an explanation for the updated confidence level.

API Sequencing

A menu item is added to the Import menu:
  • Import an OpenAPI File

Alert Review

A right click menu item "Review Using LLM/AI" that is enabled for a single alert or an alert tree.
