Options

Model Provider

Select the model provider you want to use for your LLM requests. The current supported model providers are listed below:

  • Ollama
  • Azure OpenAI

API Key

The API key is required to access your LLM API service. For Azure OpenAI, this value can be found in the "Keys and Endpoint" section when examining your resource from the Azure portal. You can use either KEY1 or KEY2.

Endpoint Base URL

The endpoint base URL is the root URL for all API requests to your LLM API service. For Azure OpenAI, this value can be found in the "Keys and Endpoint" section when examining your resource from the Azure portal. You can also find the endpoint via the "Deployments" page in the Azure AI Foundry portal. An example endpoint is: https://docs-test-001.openai.azure.com/.

LLM Model Name

The LLM model name specifies which language model you want to use for your requests.
